Exocortex shape Frozen-experimental contract

Public evidence core. Private consumers.

Exocortex is a local-first context evidence core for agents. It ships reusable machinery for sources, profiles, citations, receipts, and packets while private consumers keep roots, commands, policies, secrets, and archives outside the public repo.

Work contract

Deliverable: scrub the public/private boundary, close the shape questions as decisions, and define the retrieval experiment.

Proof now: repo-wide boundary grep, markdown/HTML inspection, experiment doc inspection, and no runtime scaffold.

Stop condition: no implementation until the retrieval experiment reports an accepted result.

Chosen design

One evidence core with generic adapters, LanceDB local index state, profiles, fetchable citations, JSONL receipts, and packet compilation.

Why it wins

It is more auditable than ad hoc Markdown without turning into a private workspace, hosted memory product, scheduler, or agent runner.

Proof surface

The next proof is a Crucible-graded retrieval experiment on private data. Build work waits for that result.

API and SDK

Mirror the core verbs instead of creating a second product: init_workspace, list_sources, ingest_source, search_context, build_context_packet, fetch_source, and explain_packet.

Experiment gate

The repo stays frozen until docs/experiments/retrieval-validation.md is run on private data and reports an accepted result. The experiment compares an incumbent private retrieval path, exo search, and exo packet with deterministic citation checks plus a Crucible relevance/completeness jury.
