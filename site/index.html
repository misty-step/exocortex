EXPERIMENTATION-OPEN, PRODUCTIZATION-GATED

Exocortex is the local-first evidence core that lets an agent ask for context and a human audit exactly what it saw.

Agents need smaller, safer context than "search everything and hope." Exocortex compiles policy-filtered, replayable context packets with fetchable citations, freshness labels, source decisions, exclusions, and gaps — so a human can inspect exactly what an agent read, what it skipped, and why, without trusting a prompt convention. There is no working exo binary yet: this site shows the real proof-of-shape fixture and the gate that keeps it honest, not a shipped product.

Built from Aesthetic tokens, local assets, and GitHub Pages.

Why it matters

job Compile a context packet an agent trusts and a human can audit. proof Every claim carries a fetchable citation and a checksum.
shape One core, projected through CLI, MCP, HTTP API, and SDKs. trust Local-first. Deterministic code owns privacy and source decisions.
status Experimentation-open, productization-gated — no `exo` binary yet. fit Public core; private consumers own their own commands and secrets.
